# Logo Types — SVG Reference Patterns

## Wordmark
Brand name typographically styled, no icon.
Good examples: FedEx (negative space arrow), NASA, Google
Rules: controlled letter-spacing, never system font, custom ligatures when possible

## Lettermark / Monogram
1 to 3 initials, optional geometric container shape.
Good examples: IBM, GE, Chanel (interlocked CC)
Rules: optical centering (not mathematical), balanced visual weight

## Pictorial
Narrative shape or stylized concept representation.
Good examples: Apple, Twitter bird, Nike Swoosh, WWF panda
Rules: recognizable at 16px, only one possible reading, no ambiguity

## Combined Logo
Icon + wordmark composition.
Rules: icon/text spacing = icon height × 0.2, optical center alignment

## Abstract / Geometric
Non-figurative shape, identity through pure geometry.
Good examples: Mitsubishi, Adidas, Pepsi
Rules: golden ratio or phi proportions, max 3 primary shapes

## UI Icon / Interface
For apps, dashboards, interfaces. Stroke-based, strict 24px grid.
Standards: Lucide (1.5px stroke), Heroicons (24px grid)
Rules: optical corrections on diagonals, minimum 1.5px stroke

## Favicon 32x32
Ultra-simplified logo version.
Rules: single strong visual element, max 2 colors, no thin strokes
Test: unreadable at real 16px = restart with fewer elements
